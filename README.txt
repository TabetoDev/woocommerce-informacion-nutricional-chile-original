Woocommerce - Información Nutricional Chile

Descripción

Este plugin permite a los dueños de tiendas en WooCommerce agregar información nutricional a sus productos, cumpliendo con la normativa chilena. Es ideal para negocios de alimentos y suplementos que necesitan mostrar tablas nutricionales detalladas en sus productos.

Características

Agrega fácilmente tablas de información nutricional en las fichas de productos.

Cumple con las normativas chilenas vigentes.

Interfaz amigable para la administración de datos.

Compatible con WooCommerce 3.0+ y WordPress 5.0+.

Compatible con PHP 7.4+ y 8+.

Requisitos

WordPress 5.0 o superior.

WooCommerce 3.0 o superior.

PHP 7.4 o superior.

Instalación

Desde el Panel de WordPress

Accede a tu panel de administración de WordPress.

Ve a Plugins > Agregar nuevo.

Haz clic en Subir plugin y selecciona el archivo .zip del plugin.

Presiona Instalar ahora y luego Activar.

Instalación Manual

Descarga el archivo del plugin desde GitHub.

Extrae el contenido en la carpeta wp-content/plugins/ de tu instalación de WordPress.

Ve a Plugins en el panel de administración y activa "Información Nutricional Chile".

Uso

Agregar Información Nutricional a un Producto

Accede a Productos en el panel de WooCommerce.

Edita un producto existente o crea uno nuevo.

En la sección "Información Nutricional", ingresa los datos requeridos.

Guarda los cambios y revisa la página del producto para ver la tabla nutricional.

Personalización

Puedes modificar los estilos de la tabla nutricional agregando reglas CSS en tu tema.

Si eres desarrollador, puedes extender la funcionalidad del plugin usando los filtros y hooks disponibles.

Actualizaciones

Para recibir actualizaciones automáticas:

Descarga la última versión desde GitHub.

Sustituye los archivos en wp-content/plugins/nut/.

Alternativamente, sube el nuevo archivo .zip desde Plugins > Agregar nuevo.

(Se recomienda habilitar un sistema de actualizaciones automáticas. Para más información, consulta la documentación oficial de WordPress o plugins de gestión de actualizaciones).
